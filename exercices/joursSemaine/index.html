AJAX - Quel est le jour de la semaine?

Énoncé de l'exercice

Sur une page, vous placez trois éléments :

  1. un input text
  2. un bouton « lire »
  3. une table avec une seule colonne dont le titre est « Valeurs »

Dans l'input, vous pouvez placer un chiffre compris entre 1 et 7 (jour de la semaine). Vous devez veiller du côté client que la valeur encodée respecte ces conditions. Lorsque vous cliquez sur le bouton, vous ajoutez à la table la valeur JSON retournée par un script PHP via AJAX. Cette valeur est le nom du jour en français correspondant au chiffre passé en paramètre. Une vérification doit également être faite du côté serveur et si le jour demandé n'est pas compris entre 1 et 7, le script retourne le chiffre demandé suivi de « n'est pas un numéro de jour valide ».
